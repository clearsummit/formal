# Installation

In order to help you understand better how this project works and why exist 3 different packages, we will start by installing the **core** package. With your development server started, open a new terminal window and run:

```sh
yarn add @kevinwolf/formal
```

## Wrapping up

On this step we just [installed the `@kevinwolf/formal` library](https://github.com/kevinwolfcr/formal-tutorial/commit/daf4bb8edf6a9a54d540d83f639a407b0da378cf#diff-b9cfc7f2cdf78a7f4b91a753d10865a2R9).
